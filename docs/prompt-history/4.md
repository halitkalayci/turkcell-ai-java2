## Refactor Planı: product-service (Contract-First Uyum)

### Tespit Edilen Problemler

**1. Base path yanlış**
Mevcut: `@RequestMapping("/products")`
Olması gereken: `@RequestMapping("/api/v1/products")`

**2. Path variable tipi yanlış (3 endpoint)**
Mevcut: `@PathVariable int id`
Olması gereken: `@PathVariable UUID id` (spec: `format: uuid`)

**3. Tüm DTO'lar eksik**
Spec'te tanımlı olup implementasyonda hiç bulunmayan sınıflar:
- `Product`
- `ProductCreateRequest`
- `ProductUpdateRequest`
- `ProductPage`
- `ErrorResponse`
- `StockUpdateRequest`
- `StockUpdateResponse`

**4. Response tipleri yanlış**
Tüm endpoint'ler `ResponseEntity<String>` döndürüyor. Her birinin spec'e göre ilgili DTO'yu döndürmesi gerekiyor.

**5. POST 200 dönüyor, 201 dönmeli + `Location` header eksik**
Spec: `"201": Created` + `Location` header zorunlu.

**6. DELETE 200 dönüyor, 204 dönmeli (body yok)**
Spec: `"204": Deleted`

**7. `GET /api/v1/products` query parametreleri eksik**
Spec: `page` (int), `size` (int), `q` (string) parametreleri + `ProductPage` response yapısı yok.

**8. `PUT /api/v1/products/update-stock` endpoint'i hiç yok**
Spec'te tanımlı olan bu endpoint controller'da mevcut değil.

**9. Bean Validation yok**
`spring-boot-starter-validation` bağımlılığı pom.xml'de eksik. `@Valid`, `@NotBlank`, `@Positive` vb. annotation'lar kullanılmıyor.

**10. Global hata yönetimi yok**
Spec `ErrorResponse` (`code`, `message`, `details`) formatı tanımlıyor. `@ControllerAdvice` ile bu formata dönüştüren bir exception handler yok.

**11. Swagger UI yok**
AGENTS.MD 2.3: "Tüm servisler Swagger implementasyonu yapmalıdır." `springdoc-openapi-starter-webmvc-ui` bağımlılığı pom.xml'de eksik.
